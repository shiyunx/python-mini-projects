## Temperature Converter
This script convert temperature from Celsius (<code>°C</code>) to Fahrenheit (<code>°F</code>) or from Fahrenheit (<code>°F</code>) to Celsius (<code>°C</code>).

    # Celsius
    °C = (°F - 32) * 5/9

    # Fahrenheit
    °F = (°C * 9/5) + 32
